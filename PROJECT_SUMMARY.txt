PROJECT HUB - MERN STACK APPLICATION
=====================================

OVERVIEW
--------
A full-stack project management application built with MongoDB, Express, React, and Node.js. 
Developers can centralize and manage all projects in one place with a modern, intuitive interface.

TECH STACK
----------
Frontend: React 18, Vite, Tailwind CSS | Backend: Node.js, Express.js | Database: MongoDB/Mongoose

KEY FEATURES
------------
• Complete CRUD operations for projects
• Modern UI with gradient designs and glassmorphism effects
• Search functionality by title, category, and description
• Status filtering (Stable, Debugging, Broken)
• One-click access to project links and GitHub repos
• Copy local commands to clipboard
• Responsive design for all devices
• Real-time UI updates after database operations

DATABASE SCHEMA
---------------
Projects include: title, description, link, category, status, techStack, githubUrl, 
localCommand, lastOpened timestamp, with automatic timestamps.

API ENDPOINTS
-------------
GET/POST /api/projects | PUT/DELETE /api/projects/:id | PATCH /api/projects/:id/last-opened

SETUP
-----
1. Install dependencies (frontend & server)
2. Configure MongoDB in server/.env
3. Run backend: cd server && npm run dev
4. Run frontend: npm run dev

This application streamlines project management for developers, providing quick access to all 
project information and links in a single, beautifully designed interface.
